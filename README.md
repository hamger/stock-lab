# stock-lab
my stock lab with Python

## structure
```
. stock-lab
├── requirements.txt  ----------------- 依赖配置
├── README.md  ------------------------ 说明文件
├── codes.csv  ------------------------ A股代码（除创业板）
├── codes2.csv  ----------------------- A股代码
├── README.md  ------------------------ 说明文件
└── lab  ------------------------------ 源码目录
```

## setup
1. 请确认已经存在 Python 3.x 环境

2. 在项目根目录执行命令: `pip install -r requirements.txt`

## usage
```
cd lab
```
```
python main.py
```
